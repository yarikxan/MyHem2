# MyHem2

funny gem for filechecking

## Installation
clone, build, install

## Usage

require, MyHem2.checkfile(fn:string) -> format:string / nil(if unknown format) / error if notexist

formats: jpg, png, avi, wav, psd, mp3, rar, zip, pdf, tar, mp4
